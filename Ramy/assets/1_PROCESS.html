What are the steps in  turning an image into data?

1. Input Layer:

 - The process starts with an input image

- The image is treated as a matrix of pixel values

2. First Convolution Layer:

 - Filters scan the image for basic features.

- Create feature maps highlighting edges, colors, and shapes.

3. First Subsampling (Pooling):

 - Reduce feature map size to focus on key features.

- Simplify data and improve efficiency.

4. Second Convolution Layer:

 - Combine simple features into complex patterns.

- Create higher-level feature maps.

5. Second Subsampling:

 - Further reduce dimensions.

- Focus on the most relevant details.

6. Fully Connected Layer:

 - Combine all features into an output vector.

- Produce the embedding representing the image.
