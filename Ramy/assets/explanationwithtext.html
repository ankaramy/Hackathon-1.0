Image Embedding Using CNNs

Step 1: Input Image
The process begins with the raw input image, represented as a grid of pixel values.
This is the starting point for the CNN to analyze and prepare the image for feature extraction.
Step 2: Initial Convolution
The CNN applies convolutional filters to the input image, capturing basic features like edges and textures.
These filters create the first set of feature maps by identifying low-level patterns.
Step 3: Deeper Convolution
As the CNN progresses, it captures more complex patterns like shapes and structures.
Each layer builds on the previous one, extracting higher-level features for a richer understanding of the image.
Step 4: Salient Regions and Pooling
Pooling operations reduce the size of feature maps while keeping the most important information.
This step allows the CNN to focus on critical areas of the image, like specific textures or structural details.
Step 5: Feature Maps and Beyond
Feature maps show how CNN filters respond to the image, highlighting specific patterns.
Early layers capture simple features, while deeper layers focus on abstract elements like architectural style, all getting vectorized in the annexed red arrow.

These embeddings cluster similar images together, enabling tasks like classification and retrieval.
